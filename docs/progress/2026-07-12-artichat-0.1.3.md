## Milestone 4: Registration, Email, And Account Security

- Registration domain normalization, optional subdomain matching, email-code/reset-token secret helpers, and additive security tables are complete in commits `7daef7f`, `3c91c5f`, and `30f3514`.

- Every send creates a delivery row, increments attempts, records sent/failed state, redacts codes and reset links from stored variables, and allows failed deliveries to be retried.

- Config now stores the email switch, SMTP connection fields, encrypted password, sender identity, public URL, and subscription-notification switch with safe defaults.
