# ArtiChat 0.1.5

0.1.5 是一次管理员后台、运营分析和模型体验优化更新。

## 更新内容

- **邮箱设置整合**：注册管理与邮箱功能统一移动到“管理员面板 -> 设置 -> 邮箱”，支持全局邮箱开关，并按开关状态管理注册验证、验证码登录、SMTP、模板和发送记录。
- **订阅运营卡片化**：订阅计划、模型权限、兑换码、礼品卡、公告和用量账本改为更易扫描的卡片布局。
- **用量账本增强**：支持按用户 ID 和用户邮箱筛选，并通过环状图查看单个模型占全部用量的比例。
- **用户概述修复**：修正用户列表中订阅、到期、角色、名称和邮箱字段的错位问题。
- **模型广场**：新增模型广场入口，展示模型运行状态、简介、访问权限、价格和近期调用量；模型编辑器支持配置展示开关和详细介绍。
- **首页菜单扩展**：管理员可配置首页自定义菜单按钮、跳转地址和内置图标。
- **更新中心独立化**：版本检查、部署状态、版本说明和重试操作统一移动到“管理员面板 -> 设置 -> 更新”，移除全局更新弹窗。
- **官方公告**：更新中心从固定的 `https://artichatupdate.artivis.cc/index.json` 获取一条结构化公告，并安全显示公告类型、日期、标题和纯文本正文。

## 兼容与安全

- 公告接口只接受 HTTPS JSON 对象，限制响应大小、字段长度和公告类型，不渲染远端 HTML。
- 版本部署继续使用健康检查、数据备份和失败回滚流程。
- 本版本不新增 Python、系统或 npm 依赖。
